In order to generate an automated uncertainty sampling report, do the following steps:

1. create empty folder where you want to be the results saved in
2. copy main_template.tex and setupStudy_template.m (rename)
3. copy the DICOM imported patient as a .mat file into the directory
4. use apropriate parameters for uncertainty sampling
5. start matlab, add matRad to the search path and run setupStudy.m

after succesfull calculation a pdf report will be saved (report/main.pdf)
