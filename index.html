matRad

An open source multi-modality radiation treatment planning sytem

Download this project as a .zip file Download this project as a tar.gz file

matRad is an open source software for radiation treatment planning of intensity-modulated photon, proton, and carbon ion therapy. matRad is developed for educational and research purposes; it is entirely written in MATLAB.

Features

matRad comprises a set of function modeling the entire treatment planning workflow as well as example patient data ans base data for all required computations. Specific modules include

  • Ray tracing
  • Photon dose calculation
  • Proton dose calculation
  • Carbon ion dose calculation
  • Inverse planning
  • Multileaf collimator sequencing

matRad is constantly evolving. If you are looking for a special feature do not hesitate and get in touch.

Development team

The development is driven by the research group Optimization Algorithms within the Division of Medical Physics in Radiation Oncology at the German Cancer Research Center in Heidelberg but we also have other researchers contributing to our developments.
